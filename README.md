# Frogger Game
A classic arcade-style Frogger game implemented in Java using JavaFX. Guide the frog across busy roads and dangerous rivers to reach the safety of its home.

# Features
Move the frog with arrow keys (↑ ↓ ← →)
Avoid moving vehicles on the road
Jump onto floating logs and turtles to cross the river
3 progressively difficult levels
Score system: 150 points per forward lane, 500 points per frog home, bonus for level completion
30-second timer with life penalties
Pause (P) and menu navigation (N, M, A, H, X)

# Technologies
Java 11+
JavaFX
Custom game engine library (provided by UCD COMP2011J)

# How to Run
Import the project into IntelliJ IDEA and run `fg.display.ApplicationStart`.
